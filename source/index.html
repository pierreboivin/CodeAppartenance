Pertinence d’un code de citoyenneté

« Pertinence : Qui est exactement adapté à l’objet dont il s’agit »

Depuis quelques temps, le conseil de bande de la nation huronne-wendat a apposé le mot « gouvernement » sur sa façade. Un terme lourd de sens, indiquant une réflexion longuement mûrit, préparant l’entrée vers un nouveau système : l’autonomie gouvernemental et l’abandon de la loi sur les indiens. Par les consultations, le gouvernement de la nation huronne-wendat semble avoir réussi à faire accepter à la population qu’avoir un code de citoyenneté est nécessaire. Généralement, on nous demande alors : « Comment le voulez-vous ?  Qu’aimeriez-vous que l’on inclus dans ce code?» Mais, sans doute, les premières vraies questions qui se posent aujourd’hui, ne serait-ce pas plutôt la suivante : « Voulez-vous vraiment d’un code de citoyenneté? Quelles en sont les réelles implications? Y-a-t’il d’autres solutions?»
